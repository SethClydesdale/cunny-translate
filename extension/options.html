Cunny Translate Options

Automatically encodes or decodes Cunny Code when text is entered into the input field.
Automatically copy and paste selected text into the input field when you open the popup.
Keeps the password checkbox ticked or unticked depending on your preference.
(If disabled, the password checkbox defaults to off every time you open the popup.)
Saves your most recent password so you don't have to type it again when encrypting or decrypting.
(The saved password is deleted upon disabling this option.)
Shows Cunny Translate in the context menu whenever you right click while text is selected.
